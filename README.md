# 📊 Tableau Retail & Sales Dashboard Project

## 🧾 Overview
This project was completed as part of a **Data Technician Bootcamp**, focusing on developing practical Tableau skills to analyse and visualise data effectively. The goal was to transform raw retail and sales data into **meaningful, interactive dashboards** that support decision-making through clear data storytelling.

## 🎯 Objectives
- Analyse retail and sales data to uncover trends and insights  
- Design visually engaging and interactive dashboards  
- Apply best practices in **data visualisation and storytelling**  

## 🛠️ Tools & Technologies
- **Tableau Public**
- Data visualisation techniques
- Calculated fields and aggregations

## 📈 Key Features
- ✅ **Interactive Dashboards**  
  Built dynamic dashboards that allow users to explore data through intuitive controls.

- ✅ **Filters & Slicers**  
  Implemented dashboard-level and worksheet-level filters to enable drill-down analysis and user-driven exploration.

- ✅ **Calculated Fields**  
  Created custom metrics and calculations to enhance analysis and provide deeper insights.

- ✅ **Variety of Visualisations**  
  Designed multiple chart types to represent data appropriately:
  - Bar charts (comparisons)
  - Line charts (trends over time)
  - Pie charts (proportions)
  - Maps (geographical insights)

- ✅ **Dashboard Integration**  
  Combined multiple worksheets into a single cohesive dashboard with consistent design and usability principles. ## 📊 Data Storytelling Approach
A strong emphasis was placed on **data storytelling**, ensuring that each visualisation contributes to a clear narrative. The dashboards were designed to:
- Highlight key sales and retail performance metrics  
- Identify trends and patterns over time  
- Provide actionable insights for business stakeholders  
- Enable quick understanding through intuitive layout and visual hierarchy  

## 🔍 Insights & Outcomes
Through this project:
- Patterns in sales performance and customer behaviour were identified  
- Visual exploration made it easier to compare regions, products, and time periods  
- Interactive elements allowed users to customise their analysis  

## 🚀 Key Skills Demonstrated
- Data cleaning and preparation  
- Visual design and dashboard layout  
- Analytical thinking and insight generation  
- Tableau functionality including:
  - Filters and parameters  
  - Calculated fields  
  - Multi-chart dashboards  

## 📌 Conclusion
This project showcases the ability to turn complex retail and sales datasets into **clear, interactive, and insightful dashboards**. It demonstrates a solid foundation in Tableau and highlights the importance of **effective data storytelling in analytics**.
